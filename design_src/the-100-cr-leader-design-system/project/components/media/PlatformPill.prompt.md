`PlatformPill` is the "Listen now" row that links out to each podcast platform - used in the hero platform bar and episode cards.

```jsx
<PlatformPill platform="spotify" tone="dark" />
<PlatformPill platform="google" tone="light" action="Listen now" />
```

`tone="dark"` for forest backgrounds, `tone="light"` for paper. Pairs with `PlatformIcon`.
