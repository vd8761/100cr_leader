`DisplayHeading` is the Bebas Neue showpiece type - the giant "THE 100 CR LEADER" headline.

```jsx
<DisplayHeading size="hero">THE 100 CR LEADER</DisplayHeading>
<DisplayHeading size="stacked" color="var(--paper)">THE<br/>100 CR<br/>LEADER<br/>CIRCLE</DisplayHeading>
```

`hero` stretches one line edge-to-edge; `stacked` is the multi-line Circle block. Always uppercase, sand by default on forest.
