`SectionLabel` is the quiet section caption - light-weight Onest with an optional mic glyph and hairline rule.

```jsx
<SectionLabel rule>Featured Episodes</SectionLabel>
<SectionLabel icon={null} tone="dark">Built From Real Business Journeys</SectionLabel>
```

`tone="dark"` for forest sections. Set `rule` to draw the divider line beneath (as in the episodes header).
