`Button` is the editorial pill CTA. Quiet by default; fills/darkens on hover.

```jsx
<Button variant="solid">Join the Circle</Button>
<Button variant="onDark" icon="arrow-right">All Episodes</Button>
<Button variant="outline" href="#book">Read more</Button>
<Button variant="ghost" icon="arrow-right">Listen now</Button>
```

Variants: `solid` (forest on light), `onDark` (sand on forest), `outline`, `ghost`. Sizes `sm`/`md`/`lg`. Pass `icon` (any `IconName`) and `iconRight`.
