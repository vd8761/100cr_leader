`Icon` renders any brand or UI glyph used across the 100 CR Leader system - fully inline, so it works at any bundle depth.

```jsx
<Icon name="spotify" size={28} color="var(--white)" />
<Icon name="google-podcasts" size={28} />
<Icon name="mic" size={20} color="var(--sand)" />
```

Names: `spotify`, `youtube`, `google-podcasts` (multicolor), `whatsapp`, `facebook`, `linkedin`, `x`, `instagram`, `mic`, `play`, `pause`, `arrow-right`. Mono marks honor `color`; `google-podcasts` keeps its Google palette.
